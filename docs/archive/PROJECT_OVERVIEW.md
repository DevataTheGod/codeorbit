# 🎓 CodeOrbit — Project Overview (Archived)

CodeOrbit is an educational software-as-a-service (SaaS) platform built to simulate a guided, hands-on engineering apprenticeship. It integrates a learning management dashboard, Socratic AI mentorship, and an in-browser IDE to keep students engaged while ensuring they actually write their own code.

---

## 🚀 Core Value Proposition

In online software education, students often fall into two traps:
1. **Tutorial Hell:** Passively watching video lessons without coding, leading to zero retention.
2. **AI Copypasta:** Asking generic chatbots (like ChatGPT) to write the code for them, copying it, and completing projects without understanding.

**CodeOrbit solves this through three pillars:**
- **Socratic AI Guidance:** The AI tutor — **Orbit** — never writes code. It only gives hints, asks questions, and validates student understanding.
- **Cheat Detection Telemetry:** Keystroke timings, copy-pastes, and inactive intervals are tracked. Copy-pasting large blocks of code flags the system, locking the milestone.
- **Proof-of-Work Oral Exam:** When flagged, the student must pass an oral code assessment with Orbit, explaining the changes they pasted. Failure flags the task for human mentor reset.

---

## 👥 Roles & Target Audience

CodeOrbit operates with three distinct user roles:

1. **Student:** 
   - Submits project ideas.
   - Follows an AI-generated milestone roadmap.
   - Codes inside the in-browser IDE with Orbit AI integration.
   - Submits milestones for mentor review.
   
2. **Mentor:**
   - Overlooks assigned student project submissions.
   - Reviews automated Socratic chat summaries and AI-generated mentor reports.
   - Grades milestones, provides feedback, and responds to help flags.

3. **Admin:**
   - Manages global user roles (Admin, Mentor, Student).
   - Monitors platform-wide analytics and performance reports.
