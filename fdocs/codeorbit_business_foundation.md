Build the essential learning infrastructure for technical training organizations where engineering judgment is measured, mentors scale, and every student becomes an independent software engineer.

## SECTION 2: THE PROBLEM

### The Core Pain (Bootcamp Perspective)

**Problem 1: Understanding Is Unverifiable**
Today, a bootcamp cannot reliably tell if a student actually understands their submitted work.

Current workflow:
```
Student submits project
↓
Mentor reviews code (30 mins–2 hours)
↓
Mentor makes assumptions about understanding
↓
Mentor gives feedback
↓
Student passes/fails in interview (4-12 weeks later)
↓
Bootcamp realizes too late if understanding was real
```

The gap between project submission and interview outcome is 4–12 weeks. By then, the student is lost, and the bootcamp cannot course-correct.

**Problem 2: AI Is Making Assessment Worse**
Students now have access to:
- ChatGPT (generates complete solutions)
- GitHub Copilot (generates code)
- Cursor/Claude (AI-powered IDE)

Result:
- Mentor cannot trust code submission
- Fake project completion rates
- False confidence in student progress
- Interview failures that bootcamp didn't predict

**Problem 3: Mentors Don't Scale**
A mentor can realistically handle:
- 20 students = Sustainable
- 50 students = Difficult
- 100+ students = Impossible

Bootcamp options:
- Hire more mentors (destroys margins)
- Reduce mentor time (students get worse feedback)
- Use generic tools (students feel ignored)

### The Impact on Business

| Metric | Impact | Cost |
|--------|--------|------|
| **Job Placement Rate** | Students fail interviews despite "completing" projects | 5–10% placement rate drop |
| **Mentor Burnout** | High mentor turnover, quality degradation | ₹10–20L+ in hiring/training costs/year |
| **Reputation Risk** | Bootcamp blamed for non-employable graduates | Lost revenue, bad reviews, lower enrollments |
| **Margin Compression** | More mentors needed, higher costs | 30–40% margin erosion |

### Why Now?

Three recent changes make this solvable now:

1. **AI Tools Demand New Assessment Methods**
   - Code ownership verification is broken
   - Need behavioral/cognitive signals, not code analysis alone

2. **Browser IDEs Matured**
   - Monaco Editor is enterprise-grade
   - Remote learning is normalized post-COVID
   - Setup friction is solved

3. **Behavioral Telemetry Is Feasible**
   - You can now measure typing patterns, paste events, code explanations
   - This creates defensible "understanding score"

## SECTION 3: THE SOLUTION

### CodeOrbit: What It Does

CodeOrbit is a **Learning Infrastructure Platform** for bootcamps that:

**1. Generates Structured Roadmaps**
- Student describes goal: "Build a SaaS app"
- System generates: Milestone 1 → Milestone 2 → Milestone 3 → Tasks
- Removes ambiguity, gives students a clear path

**2. Provides an AI Mentor (Orbit)**
- Answers student questions using Socratic method
- Asks: "What do you think should happen here?"
- Refuses: "Here's the code"
- Forces students to think, not copy-paste

**3. Measures Understanding at Scale**
- Understanding Score = Engagement + Explanation + Progress
- Shows: Which concepts each student mastered
- Alerts: Which students are at risk
- Provides: Actionable insights for mentors

**4. Detects AI-Assisted Learning**
- Paste detection (how much code was pasted)
- Reflection challenges (student must explain work)
- Code explanation scoring (do they really understand?)
- Creates audit trail of genuine learning

**5. Empowers Mentors at Scale**
- Mentor dashboard shows: All students, risk levels, concept gaps
- System summarizes: What each student struggled with
- Mentors focus on students who need help, not generic review
- One mentor can now effectively support 50+ students

### The Product Stack

**For Students:**
- Browser IDE (Monaco Editor)
- Orbit AI Mentor (Socratic method)
- Project roadmap (auto-generated structure)
- Reflection challenges (explain your code)

**For Mentors:**
- Dashboard (see all students at a glance)
- Understanding scores (which students understand what)
- Risk alerts (students likely to fail interviews)
- Conversation summaries (what each student struggled with)

**For Bootcamp Operators:**
- Cohort management
- Progress reporting
- Placement outcome tracking
- Curriculum analytics

Price based on value delivered (verifiable outcomes + mentor efficiency), not on usage (per-student initially).

**Duration:** 4–8 weeks (1 cohort)

**Student Size:** 30–50 students

**What's Included:**
- Full CodeOrbit platform access
- Orbit mentor (tuned for 2–3 tech stacks)
- Understanding score dashboard
- Mentor dashboard
- Your direct support (weekly check-ins)

**Goal:**
- Validate Understanding Score accuracy
- Measure mentor time savings (hours/week)
- Generate placement outcome data
- Build case study + testimonial

**Success Criteria:**
- Bootcamp reports 30%+ reduction in mentor review time
- Understanding score correlates with interview performance
- Bootcamp interested in full contract

**Path Forward:** Pilot success → $99–149/student annual contract

**Price:** ₹99–149 per student per cohort per year

**Cohort Size:** 50–200 students

**Billing Example:**
- 50 students × ₹125 = ₹6,250/cohort

**What's Included:**

- Orbit mentor (expanded to 5+ tech stacks)
- Understanding score (improved algorithms)
- Mentor dashboard (custom views)

- Monthly outcome reports

**Why Per-Student Works:**
- Easy to understand (bootcamps think in student numbers)
- Scales with their business growth
- Natural upsell: more students = more revenue for us
- Predictable for customer: budget per cohort = straightforward

---

#### **Tier 3: Enterprise (Scale)**

**Price:** ₹250–500 per student + outcome bonus

**Cohort Size:** 200+ students, multiple cohorts/year

**Billing Model:**
- Base: 200 students × ₹350 × 2 cohorts = ₹140,000 ARR

**What's Included:**
- Everything in Tier 2

**When You Offer This:**
- Bootcamp has 3+ cohorts running on CodeOrbit
- Data proves 15%+ placement improvement
- Bootcamp wants deeper integration
- Bootcamp scaling across regions

**Duration:** 4–8 weeks (typically one cohort cycle)

**Pricing:** ₹2.5–5L fixed fee

**Students:** 30–50 participating

**Mentors:** 2–3 lead mentors from bootcamp

**Your Commitment:**
